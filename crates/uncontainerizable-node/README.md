# @uncontainerizable/native

Native bindings for [`uncontainerizable`](https://www.npmjs.com/package/uncontainerizable): a supervisor library for
programs that can't be put in real containers.

> [!NOTE]
> This package is an implementation detail. It ships the prebuilt
> `.node` binary and the raw napi-rs surface; the ergonomic,
> documented API lives in [`uncontainerizable`](https://www.npmjs.com/package/uncontainerizable).
> Install that instead.

```sh
npm install uncontainerizable
```
